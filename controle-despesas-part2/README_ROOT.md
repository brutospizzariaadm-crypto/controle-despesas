# Controle de Despesas (MVP - Parte 02)

Projeto em Java puro. Esta versão implementa um MVP funcional para cadastro e conciliação de despesas.

## Como executar
1. Certifique-se de ter JDK 11+ instalado.
2. Compile todos os arquivos Java em src (exemplo abaixo):
   javac -d out src/main/java/br/uninga/controledespesas/**/*.java src/main/java/br/uninga/controledespesas/*.java
3. Rode:
   java -cp out br.uninga.controledespesas.Main

Arquivos de dados ficam na pasta `data/`.
